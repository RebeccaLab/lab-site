## 開源專案

- **[xLSTM-genre-classifier](https://github.com/ericodle/xLSTM-genre-classifier)**: Music Genre Research Project (MGRP) - A comprehensive music genre classification framework supporting multiple model architectures (FC, CNN, LSTM, GRU, xLSTM, Transformer, VGG16, ViT) with tools for data processing, training, evaluation, and analysis.

- **[GINSA](https://github.com/ericodle/GINSA)**: GbIf Next-gen Sequence Analyzer - A tool for extracting small ribosomal subunit (SSU) sequences from metagenomic data by pairing GBIF occurrence data with EMBL's ENA repository, enabling researchers to gather species-specific sequence data for their research.

- **[Space Cubics Website](https://github.com/spacecubics/www)**: Company website for Space Cubics, Inc. - A static site built with modern web technologies, showcasing the company's work in aerospace and technology solutions.

- **[GenreDiscern](https://github.com/ericodle/GenreDiscern)**: Music genre discriminator via artificial neural network modeling, focusing on automated music classification using deep learning techniques.

- **[Semantic-Positioning-Model-for-Japanese-Adverbs](https://github.com/ericodle/Semantic-Positioning-Model-for-Japanese-Adverbs)**: Exploration of BERT's ability to preserve semantic relationships among Japanese adverbs, combining natural language processing with fuzzy theory for nuanced linguistic analysis.
